## 2  零合并操作符 ??

解决了零合并问题，之前用的 || 不会排除 `0 '' NaN`, 而??会排除这些值，只返回 null 、 undefined两个空值

```js
let a = {b: null, c: 10}
a.b ??= 20
a.c ??= 20
console.log(a)     // 输出 { b: 20, c: 10 }
```

## 4 双位运算符

- 可以使用双位操作符来替代正数的 `Math.floor( )`，替代负数的 `Math.ceil( )`。
```js
Math.floor(4.9) === 4      // true
// 简写为：
~~4.9 === 4      // true
```
