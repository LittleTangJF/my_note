### 安装

```js
brew install nginx
```
### 启动

```js
brew services start nginx
```
### 停止

```js
brew services stop nginx
```
### 重启nginx

```js
brew services restart nginx
```
### 重新加载配置文件

```js
nginx -s reload
```
### 验证nginx配置文件是否正确

```js
nginx -t
```
### 配置文件位置


```js
/usr/local/etc/nginx/nginx.conf
```
